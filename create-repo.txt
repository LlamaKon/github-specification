========== リポジトリの作成方法 ==========


・スクラッチから作成


・既存のディレクトリをGitリポにする


・既存のリモートリポから作成



========== スクラッチから作成 ==========


# スクラッチからGitリポを作成


$ git init <project-name>


　・カレントディレクトリに.gitフォルダが入った<project-name>フォルダが作成



==========　既存のディレクトリをGitリポにする ==========


# 既存のディレクトリをGitリポにする


$ git init


　・カレントディレクトリに.gitフォルダの入ったフォルダが作成

　・既存のディレクトリをGitリポにするには、そのディレクトリに移動して上記のコマンドを実行



==========　既存のリモートリポから作成 ==========


# forkを使用（新たに開発に参加する場合など）


1. GitHubのリモートリポジトリのページを開く


2. forkボタンを押して、自分のユーザを選択し、自分のリポにコピーを作成


3. $ git clone <url>　（forkしたリポジトリをローカルリポにclone）


4. $ git remote -v　（自分のリポジトリとして紐づいているかを確認）
