========== pull & push ==========

# ローカルリポで変更した内容はリモートリポに反映　（push)


# pushの前にリモートリポの内容をローカルリポに反映　（pull)



========== pull ==========

# リモートリポの更新をローカルリポに反映

$ git pull <remote_ref> <branchname>

　・指定したリポから指定したブランチの情報を現在作業しているブランチ(アクティブブランチ）に反映

　・複数人での開発では他の人が更新をしている可能性があるため、pullする



========== push ==========

# 指定したリポの指定したブランチに、アクティブブランチの情報を送信

$ git push <remote_ref> <branchname>
